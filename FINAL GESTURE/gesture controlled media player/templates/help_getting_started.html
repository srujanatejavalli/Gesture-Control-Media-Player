Getting Started

Welcome! This guide will get you set up with Gesture Controlled Media Player quickly. For the step-by-step gesture walkthrough, go to the full Instructions page.

Quick Start

  1. Register or login to your account.
  2. Open the Player and click "Start Camera" to enable your camera.
  3. Refer to the Gesture Instructions to learn the five core gestures.
  4. Use the Feedback page to report issues or request features.

Need visual help?

Head to Instructions where each gesture is shown with an example and tips to improve recognition.
