# 🎮 Gesture Control Media Player

A web application that allows users to control YouTube videos using **hand gestures** through a webcam.
This project provides a **touchless, simple, and user-friendly** way to interact with media.

---

## 🚀 Features

* ✋ Real-time hand gesture detection
* 🎬 Play / Pause, Forward, Backward, Volume control
* 🔐 Secure login system
* 🌐 Works in browser
* ⚡ Fast and simple

---

## 🧠 How It Works

1. Webcam captures video
2. MediaPipe detects hand
3. Fingers are counted
4. Gesture is recognized
5. Action is performed on video

---

## ✋ Gesture Controls

| Gesture   | Action       |
| --------- | ------------ |
| 1 Finger  | Forward      |
| 2 Fingers | Backward     |
| 3 Fingers | Volume Up    |
| 4 Fingers | Volume Down  |
| 5 Fingers | Play / Pause |
